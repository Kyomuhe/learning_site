Polymorphism

Polymorphism means many forms and it occurs when we have many classes that are related to each other by inheritance ,inthat the same method is defined in different ways

there are two types of polymorphism in java

  1. Compile time
    • Operator overloading
    • Method Overloading
  2. Runtime
    • Method overriding

Method overriding

method overriding is a form of polymorphism which occurs in inherritance where a subclass provides a particular implementation of a method declared by its parent as in it overides the method implementation in the parent class and if the method is called the one in the subclass will be the one executed as shown below

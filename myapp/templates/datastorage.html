Data Storage

So lets talk about how we store data in java,we store our values in variables , there are two types of variables that is :

  1. Local variables :these are defined and known only within the local scope in which they are defined usually within braces { }, you will understand this better in the coming chapters
  2. Global variables : these can be accessed from any where in the class and such variables are used to represent the properties or attributes of the class

in the above example x is a variable of type int meaning it stores integers and it is assigned a value 10

Static Members

Static members also called class members are members of a class whose values belongs to a class and not the individual object of a class

static members have a word "static " after the visibility Modifier

Static means that the value belongs to all objects and when the value is changed it affects all objects

Note: class members can be accessed either through the class name or without the classname, if a member is static you can access it only inside a static context(inside a static method)

Constants

Constants are also like variables and are used to store diffrent kinds of data , but the only difference is when a value is assigned to it it can never be changed, constants are defined with the final keyword as shown below

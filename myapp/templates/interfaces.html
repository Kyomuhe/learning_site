Note: before java 8, methods in an interface were supposed to be public abstract e.g public abstract void m(); however from java 8 on wards methods in an interface can be static(and defined with a body), can be private (and defined with a body) and can be default (with the key word default before the return type)

Unlike classes that have a single root(the object class)interfaces in java donot have a single root and there for Multiple inherritance is allowed in interfaces

    points to note about interfaces:
  • by default methods in interfaces are public abstract
  • constants are by default public static final

  • We can define methods inside an interface

  • The interface should be implemented by a class, we use the keyword implements between the class and interface e.g

    public class A implements X{}

  • When a class implements an interface, it must define all the abstract methods in the interface other wise it becomes abstract
  • A class can implement more than one interace at the same time and in such a case it must define all their methods

  • if a class implements an interface and thereby overriding the interface's method, it should not lower the visibility of those methods
  • when a class implements two or more interfaces with constants of the same name but dufferent values they must be called using the interface name to avoid confusion

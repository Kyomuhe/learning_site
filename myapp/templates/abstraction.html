Abstraction

Abstraction is the process of hiding certain details and showing only essential information to the user. Abstract classes help implement Abstraction

An Abstract class is a class which is incomplete and has a keyword abstract between the visibility modifier and the keyword class e.g public abstract class A{ }

Since an abstract class is incomplete it cant be instantiated, if a class A is an abstract this statement A obj = new A(); will bring an error because abstract classes can not have objects

Note: A class which is not abstract is concrete

For a class to be abstract it must satisfy the following

  • it must be defined with the keyword abstract
  • if a class contains an abstract method i.e when a method has no body e.g public abstract void myMethod();
  • if a class extends an abstract class but doesnot define the abstract methods of the super class
  • if a class implements an interface but does not define all the abstract methods of that interface, it automatically becomes abstract

Note: A class can override a concret method of a super class and make it abstract ,this can be done if the subclass finds the super class definition invalid in its body

Note: Much s an abstract class cant be instatiated ,it can be used as a data type for an array of objects such an array of objects can never be initialised e.g if A is abstract then A[] arr = new A[5]; is correct but arr[0] = new A(); is not allowed

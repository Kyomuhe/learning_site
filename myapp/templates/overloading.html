Method Overloading

Method overloading allows a class to have more than one method with the same name but different method signature(different number of paramaters and return type)

Instead of defining two methods that should do the same thing, it is better to overload one.

in the example below the method add has been overloaded, when integers are passed when calling the method, the complier looks through the code from the beginning until a method that takes in integers is found ,now this will be the method that will be executed

this same process will be repeated if doubles are passed

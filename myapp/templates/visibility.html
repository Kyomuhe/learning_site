Visibility Modifiers

Visibility Modifiers show the level of accessibility of a class and its properties by other classes ,they are very important in herritane

We have four levels of visibilities and we use words to show this visibility

  1. public
  2. protected
  3. default
  4. private

In the table below yes means that it is accessible in that context and No means that it not accessible in that context

Visibility Modifier Accessible in that class Accessible in another class in the same package Accessible in the sub class Accessible in another package
public yes yes yes yes
protected yes yes yes No
default yes yes No No
private yes No No No

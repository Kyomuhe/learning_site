Object Casting

Objects of a class can be cast into objects of another class if both classes are related to each other through the property of inheritance

There are two types of object casting

  1. Upcasting
  2. Downcasting

upcasting/implicit:

this refers to the process of converting an object of a subclass to an object of its superclass, it is safe and automatic

Note:In upcasting the object can access only fields accessed in the super class and methods unless when the method is overridden in the sub class. but it cant access fields only defined in the sub class. in that Any additional fields or methods that are specific to the subclass are not directly accessible using an upcasted reference. However, if a method is overridden in the subclass, then when you call that method using an upcasted reference, the overridden method in the subclass will be executed, not the method from the superclass.

Generic Programming

The java generics allow us to create single class, interface and method that can be used with different types of data. This helps us reuse our code.

A class that can be used with different data types is known as a Generic class

Note: generics doesnot work with primitive types(int, float, char etc)

So lets understand the example above

public class GenericClass <T>: This line declares a generic class named GenericClass. The type parameter T is a placeholder for the actual type that will be used when creating an instance of this class

public T data;: This line declares a public field named data of type T, which will hold the data of the specified type

public GenericClass(T data): This is the constructor of the GenericClass class. It takes a parameter of type T and assigns it to the data field. This constructor allows you to initialize the data field when creating an instance of the class.

GenericClass <Integer> n = new GenericClass <Integer>(4);: This line creates an instance of GenericClass where the type parameter T is explicitly set to Integer. The constructor is called with the value 4, which is assigned to the data field. GenericClass <String> g = new GenericClass <String>("hry");: Similarly, this line creates another instance of GenericClass, but this time the type parameter is set to String. The constructor is called with the value "hry"

Inner classes

An inner class is a class that is defined within another class. It's a way to encapsulate and group related classes together, allowing them to have a more closely connected relationship. Inner classes have access to the members (fields and methods) of the outer class(classes in which they are defined), including private members, which makes them useful for implementing certain design patterns and achieving better code organization.

There are several types of inner classes in Java:

  1. Member Inner Class/Non static inner class: This is the most common type of inner class. It's defined within the body of an outer class and has access to all members of the outer class, including private members. It can also have its own members

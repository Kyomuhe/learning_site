Data types in java

    There are two categories of data types in java
  1. Primitive data types
  2. Non Primitive data types

Primitive data types are data types that are inbuilt in to the programming languange development environment , examples include

  • int -> Integers
  • double ->Floating point values
  • float-> Floating point values
  • String-> text values
  • byte -> whole numbers
  • char -> single characters

Non primitive data types are not defined in the programming languange ,they are defined by the user of the language for example classes, Arrays, and Interfaces

important point to note: all statements in java must end with a semi clone(;)other wise you will encounter a compile time error

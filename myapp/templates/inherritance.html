Inherritence

This is a concept used in object oriented programming languages where it allows you to create a new class(sub class) that inherits the properties and methods except the private ones of another class(super class)

The extends key word is used to declare inheritance between classes.Inheritance allows you to create a hierarchical relationship between classes, promoting code reusability and organization

When a class inherrits the properties of another class it can overide them and define them differently
